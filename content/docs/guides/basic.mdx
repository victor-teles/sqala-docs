In this section, we'll learn basic usage of PlaywrightPHP. If you want to see more details about playwright check the offical documentation [here](https://playwright.dev/docs/intro).

## Navigating to a page

```php
<?php

use Playwright\Playwright;

$playwright = new Playwright();
$browser = $playwright->launch();
$page = $browser->newPage();

$page->goto('https://example.com');

$browser->close();
```
