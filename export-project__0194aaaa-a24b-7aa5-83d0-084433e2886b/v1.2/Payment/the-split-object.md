## Attributes

**value** integer(>0)  
_The amount transferred to the destination recipient._

***

**recipientId** string(36)  
_The ID of the recipient._

## Example

```json
{
  "split": [
    {
      "value": 990,
      "recipientId": "17f8d060-9a43-4c2c-a4d4-d9380dab1e41",
    }
  ]
}
```
